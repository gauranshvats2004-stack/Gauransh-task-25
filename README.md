FlowApp Landing Page
Project Overview




FlowApp is a modern, responsive SaaS landing page built using HTML and Tailwind CSS.
The project showcases a clean UI with glassmorphism effects, gradient backgrounds, interactive hover animations, and a responsive bento grid layout.
The design focuses on providing an attractive user experience while highlighting the product's key features.




Technologies Used
HTML5
Tailwind CSS
CSS3 (Custom Animations)
Ionicons
